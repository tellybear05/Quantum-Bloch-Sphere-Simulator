# Quantum Bloch Sphere Simulator

A Python application for visualizing and simulating single-qubit quantum states using the Bloch sphere representation.

## Features

* Convert quantum states to Bloch sphere coordinates
* Convert Bloch sphere angles to quantum states
* Apply quantum gates (X, Y, Z, H)
* Calculate measurement probabilities
* Interactive command-line interface
* 3D Bloch sphere visualization using Matplotlib

## Technologies Used

* Python
* NumPy
* Matplotlib
* Linear Algebra
* Quantum Computing Concepts
